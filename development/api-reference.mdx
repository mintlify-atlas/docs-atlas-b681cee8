## Overview

BD-JB provides a comprehensive Java API for kernel exploitation and system interaction. All APIs are Java 1.4 compatible.

## Package Structure

```
org.bdj
├── Status                    # Output and logging
├── api/
│   ├── API                   # Low-level native interface
│   ├── Buffer                # Memory buffer operations
│   ├── NativeInvoke         # System notifications
│   └── Int8/16/32/64        # Type helpers
└── external/
    ├── Lapse                 # Main exploit chain
    ├── Helper                # System call wrappers
    ├── Kernel                # Kernel R/W primitives
    └── BinLoader             # ELF payload loader
```

### Methods

#### performSetup()

Initializes exploit environment with CPU pinning and AIO blocking.

<ParamField body="return" type="boolean">
  Returns `true` if setup completed successfully
</ParamField>

```java
public static boolean performSetup()
```

**Implementation:**
- Pins main thread to CPU core 4
- Sets realtime priority (0x100)
- Creates socketpair for blocking
- Blocks AIO worker threads
- Performs heap grooming with 0x200 AIO operations

**Example:**
```java
if (!Lapse.performSetup()) {
    Status.println("Setup failed");
    return;
}
```

#### executeStage1()

Performs double-free race condition on AIO request structures.

<ParamField body="return" type="int[]">
  Returns array of two aliased socket descriptors, or `null` on failure
</ParamField>

```java
public static int[] executeStage1()
```

**Implementation:**
- Creates 64 UDP sockets
- Establishes TCP listen socket on port 5050
- Executes up to 100 race attempts
- Uses `SO_LINGER` to delay socket close
- Returns pair of sockets pointing to same kernel memory

**Example:**
```java
int[] aliasedPair = Lapse.executeStage1();
if (aliasedPair == null) {
    Status.println("Stage 1 failed");
    return;
}
Status.println("Aliased sockets: " + aliasedPair[0] + ", " + aliasedPair[1]);
```

#### executeStage2()

Leaks kernel addresses using type confusion between `evf` and `ip6_rthdr`.

<ParamField body="aliasedPair" type="int[]">
  Socket pair from Stage 1
</ParamField>

<ParamField body="return" type="boolean">
  Returns `true` if kernel addresses were leaked successfully
</ParamField>

```java
public static boolean executeStage2(int[] aliasedPair)
```

**Leaked Addresses:**
- `kernelAddr` - Address of "evf cv" string in kernel .data
- `kbufAddr` - Address of kernel buffer
- `reqs1Addr` - Address of AIO request structure
- `fakeReqs3Addr` - Address of fake AIO batch structure

**Example:**
```java
if (!Lapse.executeStage2(aliasedPair)) {
    Status.println("Failed to leak kernel addresses");
    return;
}
```

#### executeStage3()

Performs second double-free to reclaim memory with controlled structures.

<ParamField body="aliasedSd" type="int">
  Socket descriptor from Stage 2
</ParamField>

<ParamField body="return" type="int[]">
  Returns pair of aliased pktopts sockets, or `null` on failure
</ParamField>

```java
public static int[] executeStage3(int aliasedSd)
```

**Implementation:**
- Overwrites routing header with AIO queue entry
- Executes double-free on `reqs1`
- Reclaims memory with `pktopts` structures
- Returns aliased socket pair for Stage 4

**Example:**
```java
int[] pktoptsPair = Lapse.executeStage3(aliasedPair[0]);
if (pktoptsPair == null) {
    Status.println("Stage 3 failed");
    return;
}
```

#### executeStage4()

Establishes arbitrary kernel read/write primitives.

<ParamField body="pktoptsSds" type="int[]">
  Aliased pktopts socket pair
</ParamField>

<ParamField body="k100Addr" type="long">
  Address of 0x100-sized kernel buffer
</ParamField>

<ParamField body="kernelAddr" type="long">
  Kernel base address candidate
</ParamField>

<ParamField body="sds" type="int[]">
  Socket array for pointer fixups
</ParamField>

<ParamField body="sdsAlt" type="int[]">
  Alternative socket array
</ParamField>

<ParamField body="aioInfoAddr" type="long">
  Address of AIO info structure
</ParamField>

<ParamField body="return" type="boolean">
  Returns `true` if kernel R/W was achieved
</ParamField>

```java
public static boolean executeStage4(int[] pktoptsSds, long k100Addr, 
                                    long kernelAddr, int[] sds, 
                                    int[] sdsAlt, long aioInfoAddr)
```

**Implementation:**
- Implements slow kernel read via `IPV6_PKTINFO`
- Leaks current process (`curproc`) address
- Creates worker socket for fast R/W
- Initializes pipe-based kernel R/W
- Fixes corrupt kernel pointers

### syscall()

Execute arbitrary system calls with up to 6 arguments.

```java
public static long syscall(int number)
public static long syscall(int number, long arg0)
public static long syscall(int number, long arg0, long arg1)
public static long syscall(int number, long arg0, long arg1, long arg2)
public static long syscall(int number, long arg0, long arg1, long arg2, 
                          long arg3)
public static long syscall(int number, long arg0, long arg1, long arg2, 
                          long arg3, long arg4)
public static long syscall(int number, long arg0, long arg1, long arg2, 
                          long arg3, long arg4, long arg5)
```

<ParamField body="number" type="int" required>
  System call number (e.g., `SYS_GETPID`)
</ParamField>

<ParamField body="arg0-arg5" type="long">
  System call arguments (cast pointers to long)
</ParamField>

<ParamField body="return" type="long">
  System call return value (0 on success, negative errno on error)
</ParamField>

**Example:**
```java

// Open file
Buffer pathBuf = new Buffer(256);
// ... write path to buffer
long fd = Helper.syscall(Helper.SYS_OPEN, pathBuf.address(), 0L, 0L);

// Read from socket
Buffer readBuf = new Buffer(1024);
long bytesRead = Helper.syscall(Helper.SYS_READ, (long)sockfd, 
                               readBuf.address(), 1024L);
```

## Complete Example: Custom Payload

```java
package org.bdj.external;

import org.bdj.Status;
import org.bdj.api.*;
import org.bdj.external.*;

public class MyPayload {
    public static void main(String[] args) {
        Status.println("=== My Custom Payload ===");
        
        // Check if already jailbroken
        if (Helper.isJailbroken()) {
            Status.println("Already jailbroken!");
            loadMyBinPayload();
            return;
        }
        
        // Run full exploit
        Status.println("Running Lapse exploit...");
        
        if (!Lapse.performSetup()) {
            Status.println("Setup failed");
            return;
        }
        
        int[] stage1 = Lapse.executeStage1();
        if (stage1 == null) return;
        
        if (!Lapse.executeStage2(stage1)) return;
        
        int[] stage3 = Lapse.executeStage3(stage1[0]);
        if (stage3 == null) return;
        
        // Stage 4 parameters omitted for brevity
        if (!Lapse.executeStage4(...)) return;
        
        // Post-exploitation
        Kernel.postExploitationPS4();
        
        Status.println("Jailbreak complete!");
        
        // Load custom binary
        loadMyBinPayload();
    }
    
    private static void loadMyBinPayload() {
        try {
            // Load from USB
            Buffer pathBuf = new Buffer(256);
            String path = "/mnt/usb0/mypayload.elf";
            for (int i = 0; i < path.length(); i++) {
                pathBuf.putByte(i, (byte)path.charAt(i));
            }
            pathBuf.putByte(path.length(), (byte)0);
            
            long fd = Helper.syscall(Helper.SYS_OPEN, pathBuf.address(), 
                                    0L, 0L);
            if (fd >= 0) {
                // Read and load ELF
                // ... implementation
                Helper.syscall(Helper.SYS_CLOSE, fd);
            }
        } catch (Exception e) {
            Status.printStackTrace("Payload load error: ", e);
        }
    }
}
```
